# Subdomains in Project Portfolio Management

## Overview

Subdomains classify PPM functional areas by strategic importance to guide investment decisions.

## Classification

### Core Domain

Portfolio optimization — the decision-making that determines which projects receive funding and how they align with strategy:

- Project selection and prioritization algorithms
- Strategic alignment scoring models
- Portfolio balancing across investment categories
- Scenario analysis and what-if modeling

**Investment**: Custom-built, deeply modeled, competitive differentiator.

### Supporting Subdomains

- **Project Execution Tracking**: Schedule management, milestone tracking, deliverable management
- **Resource Management**: Allocation, capacity planning, skills matching
- **Risk Management**: Risk registers, assessment frameworks, mitigation tracking
- **Financial Tracking**: Budget management, cost tracking, earned value analysis

**Investment**: Well-modeled following established PMI/PRINCE2 patterns.

### Generic Subdomains

- **Authentication and Authorization**: User login, role-based access
- **Notifications**: Email alerts, reminders, escalations
- **Document Management**: File storage, versioning
- **Reporting Infrastructure**: Dashboard frameworks, data export

**Investment**: Buy or adopt open-source.
