# Domains - Claude Configuration

This directory contains Claude AI agent configuration for the domains project.

## Files

- `AGENTS.md` - Agent instructions for working within the domains directory.
- `CLAUDE.md` - Points to AGENTS.md for Claude context loading.
- `index.md` - This file.
- `plan.md` - Project plan with goals, scope, approach, and milestones.
- `tasks.md` - Project task tracking with completion status.
- `settings.local.json` - Local Claude settings.
