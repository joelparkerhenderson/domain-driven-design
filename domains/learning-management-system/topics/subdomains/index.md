# Subdomains in Learning Management System

## Overview

Subdomains classify LMS functional areas by strategic importance.

## Classification

### Core Domain

What differentiates this LMS from competitors:

- **Adaptive Learning**: Personalized content sequencing based on learner performance
- **Assessment Analytics**: Intelligent grading, learning outcome measurement, competency mapping
- **Learning Path Optimization**: Dynamic learning paths that adjust based on progress and goals

**Investment**: Custom-built, deeply modeled, competitive differentiator.

### Supporting Subdomains

- **Course Catalog**: Course definitions, curriculum structure, prerequisites
- **Enrollment**: Registration, waitlists, cohort management
- **Content Delivery**: Module and lesson presentation, SCORM rendering
- **Progress Tracking**: Completion tracking, certificates, transcripts
- **Collaboration**: Forums, group projects, peer reviews

**Investment**: Well-modeled following established educational patterns.

### Generic Subdomains

- **Authentication/Authorization**: SSO, LTI authentication, role-based access
- **Notifications**: Email alerts, reminders, announcements
- **File Storage**: Document and media asset management
- **Video Hosting**: Video streaming and transcoding
- **Reporting Infrastructure**: Dashboard frameworks, data export

**Investment**: Buy or adopt open-source.
