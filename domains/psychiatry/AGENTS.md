# Psychiatry Domain - DDD Documentation

This domain applies Domain-Driven Design (DDD) to psychiatry systems, covering diagnostic assessment, medication management, psychotherapy, crisis management, rehabilitation services, and outcomes measurement.

## Bounded Contexts

1. Diagnostic Assessment Context - psychiatric evaluation, DSM-5 diagnosis, structured interviews, rating scales
2. Medication Management Context - psychopharmacology, polypharmacy, therapeutic drug monitoring, pharmacogenomics
3. Psychotherapy Context - therapeutic modalities (CBT, DBT, psychodynamic), session management, treatment protocols
4. Crisis Management Context - psychiatric emergencies, involuntary holds, suicide risk assessment, de-escalation
5. Rehabilitation Services Context - psychosocial rehabilitation, supported employment, housing, skill building
6. Outcomes Measurement Context - clinical outcomes, functional assessment, patient satisfaction, quality metrics
